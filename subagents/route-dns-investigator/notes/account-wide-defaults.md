# Account-wide defaults

For this Worker, keep these runtime values in Cloudflare:

ACCOUNT_SUBDOMAIN should be jaredtechfit.
DEFAULT_ZONE_NAME should be agentfeedoptimization.com.
DEFAULT_SCAN_ALL_ZONES should be true.

The code defaults scan-all-zones to true even if the last value is not present, so the runtime setting is only an explicit reminder.
